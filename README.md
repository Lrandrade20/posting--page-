# 📝 Posting Page

Projeto desenvolvido para simular a criação de uma postagem, semelhante a redes sociais como Facebook ou LinkedIn.

---

## 📘 Sobre o projeto

Esta aplicação permite que o usuário escreva um título e um conteúdo para uma postagem.  
Ao clicar no botão **Publicar**, o conteúdo é exibido na tela, simulando como ficaria em um blog ou rede social.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🧩 Funcionalidades

- ✔ Campo para título da postagem  
- ✔ Campo para conteúdo da postagem  
- ✔ Botão para publicar  
- ✔ Exibição do conteúdo digitado na tela  
- ✔ Estilização básica com CSS  
- ✔ Manipulação do DOM com JavaScript
